# openEHR Affiliate Website Template

This is designed to be a template for openEHR affiliates to use for an initial version of their web presence. It's deliberately very simple using only HTML, JavaScript and CSS and contains placeholders for them to fill in with their information.

### Inspiration

The below existing openEHR Affiliate pages were used as a source of ideas for what kind of content might be valuable to include.

- https://github.com/SevKohler/openEHRDe
- https://openehr.no/
- https://openehr.atlassian.net/wiki/spaces/SWE/pages/90796248/openEHR+Sweden
- https://openehr.nl/
- https://openehr.ch/
