# openEHR Affiliate Website Template

This is designed to be a template for openEHR affiliates to use for an initial version of their web presence. It's deliberately very simple - using only HTML, JavaScript and CSS and contains placeholders for them to fill in with their information.

### Discourse Page Publishing

Note that there is also another option for affiliates to publish their content - using the openEHR Discourse forum. This is a good option for affiliates who want to have a more interactive presence, and who are comfortable with using Discourse. The template website can be used as a starting point for affiliates who want to create a more traditional website, but the Discourse forum can be used as an alternative for those who prefer it.

- https://discourse.openehr.org/t/example-affiliate-page-publishing/11384
